**Requirements:** (Iam using Ubuntu 24.04)

- jq
- bun
- pm2
- nvm
- snap
- go

## jq

```bash
apt-get install jq
```

## NVM (Node & npm)

```bash
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash
source ~/.bashrc
nvm install --lts
```

## bun

```bash
npm i -g bun
```

## pm2

```bash
npm i -g pm2
```

## snap

```bash
apt-get install snapd
```

## go

```bash
snap install go --classic
```
